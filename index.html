Cases

  1. Intertap

    Multiplayer highly competitive mobile game.

    The gameplay essence is hardcore-battle between two players. You need to tap the screen to build up bombs and defend yourself from you rivals' bombs. The great solution for deciding arguments in a happy get-together.

    Client: Unity (C#)
    Server: Node.js, Restify, MQTT, PostgreSQL

    Behance link (design details)
    Google Play
    In collaboration with Oz Interactive

  2. WePlay

    Quiz game system to be used in cinema theatres (questions appear on a cinema screen and players/viewers answer from their phones). The system was used as part of indirect advertising campaign.

    Technologies: MEAN Stack (MongoDB, Express.js, AngularJS, Node.js)

    In collaboration with Oz Interactive

  3. Tanuki Drunk Racing

    Arcade racing game (idea's partially inspired by Japanese mythology).

    Game Jam Topic: Luck
    Game Engine: Unity
    Graphics/Modeling: MagicaVoxel

    Download for Win, Mac or Linux

    * the game was not released for PS 4
